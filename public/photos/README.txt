PHOTOS FOLDER
-------------
Drop your real route photos here, then point to them from
content/wayfinding.ts (the "photo:" value on each connection).

Tips for fast loading on the school wi-fi:
- Use JPG, around 800-1200px wide, saved at "medium" quality.
- Keep file names simple and lower-case, e.g. atrium-to-stairs.jpg
- Keep the same file name as the placeholder to swap a photo in instantly.

The files here now are coloured PLACEHOLDERS so you can see the app working.
Replace them with real photos whenever you're ready.
